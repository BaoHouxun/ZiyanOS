# ZiyanOS 桌面环境模拟系统

一个轻量级的桌面环境模拟系统，展示现代化的桌面界面和基础应用功能。

## 项目简介

这是一个基于 Qt/QML 开发的桌面环境模拟系统，实现了现代操作系统的核心桌面功能。系统包含锁屏界面、桌面环境、应用窗口管理和系统设置等完整功能。

## 主要功能

- **锁屏界面**：显示时间和日期，保护系统安全
- **桌面环境**：应用图标网格布局，任务栏管理
- **窗口管理**：多应用窗口的创建、关闭和切换
- **系统设置**：壁纸、窗口样式、分辨率等设置
- **应用集成**：浏览器、文件管理器、计算器、文本编辑器等
- **鼠标覆盖支持**：为PE环境提供鼠标指针支持

## 技术栈

- **前端**：Qt/QML
- **后端**：C++/Qt框架
- **构建系统**：CMake
- **目标平台**：Windows（主要），支持跨平台

## 文件结构

```
ZiyanOS/
├── src/                    # 源代码目录
│   ├── main.cpp           # 程序入口点
│   ├── modules/           # 功能模块
│   ├── resources/         # 资源文件（字体、图标等）
│   └── qml/              # QML界面文件
└── wallpapers/            # 壁纸文件夹
    ├── wallpapers.json    # 壁纸配置文件
    └── *.jpg              # 壁纸图片文件
```

## 特色功能

1. **壁纸管理系统**：支持系统壁纸和自定义壁纸，壁纸信息可配置
2. **窗口主题自定义**：可调整窗口标题栏颜色和模式
3. **任务栏窗口管理**：显示当前打开的窗口，支持快速切换
4. **PE环境适配**：自动检测PE环境并启动鼠标覆盖程序
5. **日志系统**：完整的应用程序日志记录
